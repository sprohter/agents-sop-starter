# Tools

> 这里登记可复用工具的索引卡。工具实现可以放在本地或私有仓，public starter 只放公开安全说明。

## 工具索引卡应说明

| 字段 | 说明 |
|------|------|
| 名称 | 工具或脚本名称 |
| 用途 | 解决什么问题 |
| 输入 | 需要哪些参数或文件 |
| 输出 | 会生成什么结果 |
| 风险 | 是否会写外部系统、删除文件或触发后台任务 |
| 配置 | 只写占位符，不写真实凭证 |

## 安全边界

- public starter 不保存真实工具凭证。
- 写外部系统的工具默认需要用户确认。
- 后台工具必须静默运行，失败写日志或状态。
- 工具输出如果是运行态、日志或临时报告，不进入 public 仓。

## 与 SOP 的关系

SOP 可以引用工具，但工具不能绕过 SOP 的验收标准和 `contract.md` 的安全边界。
