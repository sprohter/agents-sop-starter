# Collaboration

> 这里放多 agent 或多人协作的公开安全规则。

## 适用场景

- 主 agent 需要子 agent 做只读复核。
- 需要 Claude / Codex / 其他工具交叉检查方案。
- 长任务需要交接摘要或恢复上下文。
- 高风险变更需要独立 reviewer。

## 基本规则

- 子任务范围要窄，输出结论、证据、风险和建议。
- 主 agent 负责最终判断，不直接照搬 reviewer 输出。
- 交接摘要不能包含凭证、运行态或未脱敏业务证据。
- 最终答复要重新整理给人看，不输出内部 checkpoint。

## 最小交接格式

```text
目标：
范围：
已完成：
证据：
风险：
下一步：
```
